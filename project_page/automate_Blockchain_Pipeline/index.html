The Challenge

Public blockchain data is massive and unstructured. The primary objective was to build a robust pipeline that could handle half a terabyte of data natively within Google BigQuery while ensuring strict data quality and automated transformation logic.

Whale Analysis: Specific models were designed to identify transactions exceeding $1,000,000, providing real-time insights into institutional movements.

DevOps & Quality Assurance

Using GitHub Actions and Docker, the pipeline is fully automated. Every pull request triggers a virtual environment that executes dbt run and dbt test, validating unique keys and ensuring no null values enter the production marts.
